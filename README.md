📊 Data Science Project – Play Store Apps Analysis
🚀 Overview

![banner](.png)

Project ini adalah bagian dari perjalanan belajar Data Science saya menggunakan dataset aplikasi Google Play Store. Fokus utama adalah memahami pola aplikasi, rating, harga, dan faktor yang mempengaruhi popularitas aplikasi.

📁 Dataset
Sumber: Dataset Google Play Store Apps
Isi data: Nama aplikasi, kategori, rating, reviews, installs, price, dll.

🎯 Goals
Membersihkan dan memahami data (Data Cleaning)
Melakukan Exploratory Data Analysis (EDA)
Menemukan insight tentang aplikasi populer
(Opsional) Membangun model sederhana untuk prediksi.

🧹 Data Cleaning
-Menghapus missing value
-Menangani data duplikat
-Mengubah tipe data (contoh: installs, price)
-Standarisasi format data

📊 Exploratory Data Analysis (EDA)
Beberapa analisis yang dilakukan:

-Distribusi rating aplikasi
-Kategori aplikasi paling populer
-Aplikasi gratis vs berbayar
-Hubungan jumlah review dengan rating

🔍 Insight Sementara
Aplikasi gratis jauh lebih dominan
Kategori tertentu lebih banyak diunduh
Rating tidak selalu menentukan jumlah install
Banyak data “kotor” yang perlu dibersihkan sebelum analisis
